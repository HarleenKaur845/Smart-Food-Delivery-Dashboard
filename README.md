# Smart Food Delivery Dashboard

## Overview

The Smart Food Delivery Dashboard is an interactive data analytics project built using Streamlit. It helps analyze restaurant performance, customer ratings, delivery efficiency, pricing trends, and geographic insights.

## Features

* Interactive Home Page
* Dataset Explorer with Filters
* Restaurant Performance Analysis
* Customer Ratings Analysis
* Delivery Time Insights
* Cost & Pricing Analysis
* Geographic Insights
* Business Recommendations

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* Matplotlib
* Seaborn

## Dataset Columns

* ID
* Area
* City
* Restaurant
* Price
* Average Rating
* Total Ratings
* Food Type
* Address
* Delivery Time
